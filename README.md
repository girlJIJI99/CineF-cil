# CineFácil

Sistema de gestión de reservas para cine, con interfaz en consola que permite hacer, ver y cancelar reservas.

## Características

- Hacer reservas
- Ver reservas activas
- Cancelar reservas
- Menú interactivo sencillo

## Requisitos

- Python 3.x
